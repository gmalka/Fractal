# Fractal

Fractal renderer project 

# Description

School project using the 42 School mlx library to render fractals

Available arguments :

- Mandelbrot
- Julia
- Borningship

| **Key** | **Option** |
|----------------|---------|
| `+`, `-`, `MButton` | Change color |
| Arrow keys | Move fractal |
| Left mouse button(Only Julia) | Change complex parameter |
| `ESC` | Quit |
