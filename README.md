# Fractal

Fractal renderer project 

# Description

School project using the 42 School mlx library to render fractals

Available arguments :

- Mandelbrot
- Julia
- Borningship

| LEFT | LEFT |
|----------------|:---------:|
| **Key** | **Option** |
| +, -, MButton | change color |
| Arrow keys | move fractal |
| Left button(Only Julia) | Change complex parameter |
| ESC | Quit |
